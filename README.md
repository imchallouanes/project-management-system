# ERP Gestion Électromécanique

## 📌 Description
Application de gestion développée en Python pour une entreprise d’électromécanique.  
Elle permet de gérer les employés, les projets, les tâches et la facturation automatique avec une base de données intégrée.

---

## ⚙️ Fonctionnalités
- Gestion des employés  
- Gestion des projets  
- Suivi des tâches  
- Génération automatique de factures  
- Gestion des acomptes et primes  
- Stockage des données (SQLite / MySQL)

---

## 🧰 Technologies
- Python  
- SQLite / MySQL  
- Tkinter / PyQt
